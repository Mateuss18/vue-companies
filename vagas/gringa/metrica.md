Desarrollador/a Frontend Vue.js, ArcGIS - Remoto

En Métrica buscamos un/a Desarrollador/a Front - end con experiencia sólida en Vue.js para incorporarse a un proyecto de mantenimiento y evolución de aplicaciones con arquitectura moderna y fuerte componente geoespacial.

La persona seleccionada participará en el desarrollo de nuevas funcionalidades, optimización de aplicaciones existentes y mejora continua del código en un entorno técnico avanzado.

Responsabilidades del Puesto:

Desarrollo Front end:

Desarrollo de nuevas funcionalidades y evolutivos en aplicaciones SPA con Vue.js (Vue Router, Vuex/Pinia).
Diseño de componentes reutilizables y modulares.
Participación en arquitectura basada en microfrontends (integración, comunicación entre módulos y despliegues independientes).
Implementación de estilos con CSS/SCSS y frameworks como Tailwind o Vuetify.
Optimización de rendimiento y gestión eficiente del estado.

Integración de Mapas y Servicios

Integración y personalización de mapas mediante ArcGIS API for JavaScript.
Consumo de servicios REST, WMS, WFS y GeoJSON.
Implementación de herramientas de interacción y análisis espacial (buffers, intersecciones, cálculo de áreas).
Integración con APIs REST/GraphQL y gestión de autenticación/autorización.

Calidad, mantenimiento y colaboración

Aplicación de buenas prácticas de desarrollo, código limpio y documentado.
Testing (Jest, Cypress) y optimización continua.
Elaboración de documentación técnica.
Gestión de código con Git (GitHub/GitLab).
Trabajo colaborativo con equipos multidisciplinares bajo metodologías ágiles y uso de herramientas de ticketing (JIRA, Remedy).

Requisitos Imprescindibles:

Experiencia demostrable de al menos 3 años en desarrollo front-end.
Experiencia sólida con Vue.js y su ecosistema.Vue.js (versión 2 y/o 3).
Dominio de JavaScript ES6+, HTML5 y CSS3.
Experiencia con ArcGIS API for JavaScript.
Consumo de APIs REST y/o GraphQL.
Experiencia en arquitectura de microfrontends.
Conocimientos y experiencia en Angular.
Uso avanzado de Git.

Se Valorará (Requisitos Deseables no imprescindible):

Experiencia con frameworks de CSS como Tailwind CSS o Bootstrap.
Conocimiento de TypeScript.
Experiencia con herramientas de construcción como Webpack o Vite.
Experiencia con otras librerías o frameworks de JavaScript (React, Angular).
Conocimiento de bases de datos NoSQL.
