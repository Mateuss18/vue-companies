# Senior Software Engineer(VueJS)

Description

Razer is hiring a senior-level frontend engineer to build scalable web applications and contribute to a gamer-focused global ecosystem of hardware, software, and services.

Job Responsibilities

Lead the development of scalable and performant web applications. Translate wireframes and mockups into responsive and accessible UI components. Collaborate with product owners, designers, and engineers to deliver high-quality features. Participate in requirement gathering, scope definition, and sprint planning. Write clean and maintainable code using modern frontend frameworks. Develop and maintain unit and E2E tests. Lead code reviews and improve development practices. Support CI/CD workflows and deployment processes. Mentor junior engineers and help allocate engineering resources across projects.

Required Qualifications

Strong proficiency in HTML, CSS, JavaScript, and modern frontend principles. Strong hands-on experience with Vue.js, preferably v3, and component-based architecture. Experience with Nuxt.js for SSR and SPA applications. Familiarity with responsive design and mobile-first development. Skilled with CSS frameworks such as Tailwind or Bootstrap and preprocessors like SASS or LESS. Experience with build tools like Vite or Webpack. Strong knowledge of frontend testing frameworks such as Playwright and Vitest. Understanding of Git workflows and CI/CD pipelines. Strong communication skills in English and Mandarin.

Bonus Skills

Performance optimization, SEO best practices, Google Analytics, Google Tag Manager, and accessibility standards.
