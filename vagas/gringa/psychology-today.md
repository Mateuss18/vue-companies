# Full Stack Developer (Frontend Focus)

About The Company

Psychology Today is the world’s largest directory for mental health professionals, dedicated to connecting individuals with the support they need. Its platform helps more than one million people per day find therapy resources and mental health care through a modern technology ecosystem.

About The Role

Psychology Today is seeking a Full Stack Developer to join its mission-driven team. The role contributes to the development of the Therapy Directory by building and maintaining modular frontends, APIs, and backend services. It supports the platform therapists use to manage profiles and connect with clients.

Responsibilities

Develop modular frontends using Vue.js or similar frameworks within a micro frontend architecture, with federated modules integrated into a parent application. Collaborate with backend engineers to build scalable event-driven systems using AWS services such as AppSync, API Gateway, Lambda, and EventBridge. Implement performant, accessible, and reusable UI components across browsers and devices. Contribute to automated builds, testing pipelines, and CI/CD workflows. Participate in an Agile environment with two-week sprints, daily standups, planning meetings, demos, and retrospectives. Work effectively in a fully remote distributed team using tools such as Slack, Jira, and Google Workspace.

Qualifications

At least 3 years of experience in frontend or full stack development delivering polished production-ready UI. Strong expertise with Vue.js, component-based architecture, and module federation for scalable frontend solutions. Experience integrating frontends with serverless backend services, especially GraphQL via AWS AppSync and REST APIs via API Gateway. Familiarity with backend development in Golang within AWS serverless environments such as Lambda, EventBridge, and DynamoDB. Solid understanding of event-driven architecture and asynchronous workflows. Ability to work independently and communicate clearly in a remote, asynchronous-first culture.

Benefits

Flexible remote work, competitive independent contractor compensation, stable revenue-generating business, paid time off, performance bonuses, and the opportunity to improve access to mental health care at scale.
