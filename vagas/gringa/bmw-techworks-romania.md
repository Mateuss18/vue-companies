# Vue.js Developer

BMW TechWorks Romania presents the role as a frontend engineering position within BMW Group's European IT hub in Cluj. The work focuses on building and scaling web applications, reusable Vue.js components, testing strategies, state management, and responsive interfaces that support production, sales, HR, and financial services systems.

The posting asks for strong Vue.js and TypeScript experience, modern build tooling, REST and GraphQL integration, accessibility, CI/CD awareness, and collaboration with stakeholders in an agile environment. The position is hybrid and framed as part of BMW TechWorks' broader push around digitalization and mobility.
