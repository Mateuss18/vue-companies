  - Node.js
  - NestJS

- Banco de dados:
  - MariaDB / MySQL
  - Conhecimento em Views, Triggers e Procedures
- Cache e mensageria:
  - Redis
- Comunicação em tempo real:
  - WebSockets
- Processamento assíncrono:
  - Filas (BullMQ / RabbitMQ)
- Boas práticas:
  - Código limpo e manutenível
  - Clean Code, SOLID e Clean Architecture
- Qualidade de código:
  - Identificação e refatoração de Code Smells
- Testes:
  - Experiência com testes unitários
- Infraestrutura:
  - Docker
  - Kubernetes
- Cloud:
  - Azure Cloud
