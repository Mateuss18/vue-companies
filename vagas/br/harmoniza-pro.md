# Engenheiro de software

Engenheiro(a) de Software — HarmonizaPRO

Sobre a vaga

Buscamos um(a) Engenheiro(a) de Software para integrar o time técnico da HarmonizaPRO, empresa de tecnologia focada em automação, IA e infraestrutura digital para clínicas de harmonização facial. Você vai atuar na manutenção e evolução de um sistema web interno, além de contribuir com as automações e integrações que movem o produto.

Sobre a HarmonizaPRO

Somos a maior aceleradora comercial para clínicas de harmonização facial do Brasil. Fundamos a empresa em 2021, saímos de 2 founders para 50 pessoas e temos sede em Balneário Camboriú. Nosso objetivo é ser o maior ecossistema de marketing e vendas para clínicas do mundo.

O que você vai fazer

- Manter e evoluir o sistema web interno com Node.js e Vue.js
- Desenvolver e sustentar fluxos de automação em n8n
- Integrar sistemas via API, incluindo WhatsApp, LLMs e bancos de dados
- Propor melhorias de arquitetura e processo
- Implementar testes e garantir qualidade do código

Requisitos obrigatórios

- Experiência comprovada de no mínimo 1 ano em desenvolvimento de software
- Formação concluída ou em fase final em Ciência da Computação, Sistemas de Informação, Engenharia de Software ou áreas afins
- Inglês técnico para leitura de documentação
- Domínio sólido de JavaScript e TypeScript
- Node.js para APIs, integrações e backend
- Vue.js para manutenção e evolução de interfaces
- PostgreSQL para queries, modelagem e boas práticas
- n8n para construção e manutenção de workflows
- Git para versionamento, branching e code review
- Lógica de algoritmos e estrutura de dados
- Conhecimento de processos de engenharia, arquitetura, testes e CI/CD

Diferenciais

- Conhecimento de Design Patterns e aplicações práticas
- Familiaridade com Clean Architecture
- Vivência com Scrum, Kanban, TDD e code review estruturado
- Experiência com LLMs e agentes de IA
- Familiaridade com Supabase
- Contato com integrações via WhatsApp API
- Docker e ambientes auto-hospedados

Contrato e remuneração

Modelo: PJ
Remuneração: R$ 3.000 a R$ 5.000 por mês, conforme experiência
100% remoto
