# Desenvolvedor Front-End (Vue.js) – Nível Pleno

Buscamos um(a) Desenvolvedor(a) Front-End Pleno apaixonado(a) por criar interfaces fluidas e performáticas. Você atuará na evolução da nossa plataforma, utilizando boas práticas de componentização e garantindo a qualidade do código entregue. As atividades diárias incluirão a colaboração com a equipe de design e back-end para implementar soluções eficientes e de qualidade.

Principais Responsabilidades

- Desenvolver e manter interfaces reativas utilizando Vue.js
- Implementar gerenciamento de estado complexo com Pinia
- Consumir APIs RESTful e/ou GraphQL, tratando erros e estados de carregamento
- Criar componentes reutilizáveis e modulares, com Design System
- Garantir a fidelidade do layout em colaboração com o time de UI/UX
- Escrever testes unitários e de integração para garantir a estabilidade do front-end
- Realizar code reviews construtivos com o time

Requisitos Técnicos

- Experiência sólida com Vue.js 3 usando Composition API
- Conhecimento em gerenciamento de estado com Pinia
- Domínio de HTML5 e CSS3
- Experiência com Tailwind CSS
- Conhecimento avançado em TypeScript e JavaScript
- Experiência com Nuxt 4
- Experiência com Vite
- Controle de versão com Git usando Gitflow

Diferenciais

- Experiência com Figma
- Conhecimento em testes automatizados, como Jest, Vitest e Cypress
- Noções de acessibilidade web
- Experiência em otimização de performance
