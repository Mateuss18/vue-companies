# Senior Software Engineer (Front-end)

R&D – São Paulo, SP (Remoto)

Como Senior Software Engineer, você será responsável por desenvolver, escalar e manter novas versões de um dos produtos. Atuação fullstack com foco principal em Front-end, assumindo demandas de Back-end quando necessário para garantir entregas ponta a ponta. Trabalho em parceria com times de desenvolvimento para desenhar software e APIs e implementar melhorias contínuas para clientes. Reporte direto ao Team Lead de R&D.

## Responsabilidades
- Colaborar com arquitetos e PMs para requisitos técnicos, viabilidade e escalabilidade
- Projetar e manter arquitetura de front-end, com reuso de componentes e manutenibilidade
- Apoiar sustentação, manutenção e desenvolvimento de serviços back-end (Java), integrando RabbitMQ e modelando dados
- Definir e implementar contratos de APIs REST para integrações eficientes
- Aplicar boas práticas (SOLID, KISS, Clean Code)
- Contribuir via code reviews e compartilhamento de boas práticas
- Criar desenhos de arquitetura técnica para a interface e integração com microsserviços

## Qualificações
- 5+ anos de experiência em desenvolvimento de software
- JavaScript (ES6+) e TypeScript
- Domínio de Vue.js, HTML5, CSS3 (Sass/Less)
- Java (Quarkus é diferencial)
- Criação e consumo de APIs REST
- Mensageria (RabbitMQ)
- Git, Docker e Kubernetes
- SQL e bancos relacionais (PostgreSQL)

## Arquitetura e DevOps
- Design patterns (MVC, Strategy)
- Serviços escaláveis e microserviços/micro-frontends
- Experiência em cloud e pipelines de CI/CD

## Metodologias
- Scrum/Kanban
- Testes unitários e de integração (Jest, Vitest, Cypress ou similares)
- TDD é diferencial
