# Vue.JS Developer - Cliente (fintech)

Front-end Developer (Vue.js) – Lisboa | Tempo integral | 3 dias por semana presencial

3+ anos de experiência – €40–50K

Buscamos uma pessoa desenvolvedora front-end com forte experiência em Vue.js 3 para atuar em um time fintech em Lisboa.

## Sobre a vaga

Você contribuirá para construir e evoluir aplicações web modernas no domínio financeiro. Experiência em Vue 3 é necessária; experiência em Vue 2 é aceita. Ter atuado com React ou Angular é um diferencial. Experiência com desenvolvimento mobile é muito valorizada.

## Responsabilidades
- Desenvolver, manter e evoluir aplicações web baseadas em Vue.js
- Colaborar com back-end, design e produto para entregar features e melhorar UX
- Escrever código front-end limpo, manutenível e testável usando TypeScript
- Integrar com APIs e serviços de back-end (REST, GraphQL etc.)
- Participar de code reviews, dailies e planejamento de sprint
- Acompanhar tendências e boas práticas modernas de desenvolvimento front-end

## Requisitos
- 3+ anos de experiência prática com Vue.js em aplicações de produção (preferencialmente Vue 3)
- Domínio de JavaScript, TypeScript, HTML5 e CSS3/SASS
- Experiência com bibliotecas de gerenciamento de estado (Pinia, Vuex etc.)
- Conforto em ambientes enterprise com regras de negócio complexas
- Familiaridade com Git, CI/CD e ferramentas de build (Webpack, Vite etc.)
- Boa colaboração e compromisso com qualidade
- Inglês fluente (oral e escrito)

Requisitos adicionados pela pessoa recrutadora
- 3+ anos de experiência com JavaScript
- 2+ anos de experiência com Vue.js
- Atuação em modelo híbrido
