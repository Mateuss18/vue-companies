# Frontend Vue - R y C Consultores Asociados

A R y C Consultores Asociados está procurando desenvolvedores(as) frontend com pelo menos 3 anos de experiência para integrar um time ágil, colaborativo e focado em entregar soluções tecnológicas de qualidade. A vaga pede domínio de Vue 2 e Vue 3, Composition API e Options API, integração com APIs REST, uso de Vue Router, Vuex ou Pinia, Vite, Axios e controle de versão com Bitbucket em fluxos colaborativos.

Também são valorizadas experiência com design systems, Figma e, idealmente, desenvolvimento e deploy em Modyo, embora esse ponto seja desejável e não obrigatório. A posição é híbrida em Las Condes, no Chile, e faz parte do contexto de uma consultoria orientada a soluções de negócios baseadas em tecnologia.
