# Senior Software Engineer(.NET, Node.js, Vue.js) - Themesoft Inc.

A Themesoft está divulgando uma vaga remota para Senior Software Engineer em um cenário típico de consultoria e alocação para clientes. O papel combina frontend com Vue.js e backend com .NET e Node.js, cobrindo tanto desenvolvimento web quanto integração com APIs e serviços externos.

A vaga também destaca uso de Vue.js com Vuex, Node.js, GraphQL, testes, orientação técnica a profissionais juniores e atuação próxima de diferentes áreas para entregar soluções escaláveis e seguras em projetos conduzidos pela consultoria.
