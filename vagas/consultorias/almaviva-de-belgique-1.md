# Full Stack (Vue.js JavaScript framework) - Cliente

## Descrição da vaga

- Escrever código de front-end web usando o framework JavaScript Vue.js. Conhecimento em outros frameworks (como React) é um diferencial.
- Escrever testes unitários de front-end e back-end.
- Gerenciar a configuração de build do front-end e as tarefas npm adequadas.
- Trabalhar com TypeScript e desenvolvimento web moderno.
- Escrever CSS e HTML usando SASS.

## Back-end e microsserviços

- Participar da implementação em .NET de uma solução em microsserviços, escrevendo código de back-end de forma confiável, escalável, segura e testável.
- Implementar integrações resilientes, atendendo requisitos e SLAs.
- Implementar mecanismos de validação como testes unitários e de integração tanto no back-end quanto no front-end.
- Implementar mecanismos de autenticação OAUTH.
- Escrever e manter documentação técnica da solução, incluindo práticas de secure coding, logging e auditoria.
- Comunicar-se de forma clara e concisa com pares e com lideranças quando necessário.

## Requisitos
- 12+ anos de experiência em desenvolvimento de software
- 3+ anos de experiência em desenvolvimento front-end com Vue e TypeScript
- Experiência como front-end em 2+ projetos web em produção com Vue
- Experiência em 2+ serviços REST API em produção com ASP.NET Core e/ou .NET 8+
- Experiência em 1+ projeto em produção com testes unitários no front-end
- Experiência em 1+ projeto em produção com padrão OAUTH
- 5+ anos de experiência em desenvolvimento back-end com Microsoft .NET (C#) e .NET Core
- 3+ anos de experiência com sistemas altamente escaláveis e microsserviços
- 3+ anos de experiência trabalhando em time seguindo práticas Agile/Scrum
- 3+ anos de experiência com aplicações que publicam e consomem mensagens via AMQP (roteamento, filas e tratamento de erros)
- Conhecimento sólido de CSS e HTML para componentes web
- Bom conhecimento de componentes de back-end (REST APIs, padrões de integração, microsserviços)
- Bom conhecimento de bancos relacionais, com foco em Microsoft SQL Server (modelagem de schema, scripts etc.)
