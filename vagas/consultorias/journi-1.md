# Fullstack Developer (Vue/.Net) - UK (Mainly Remote) - Cliente (Bloxham)

Sobre
A Journi é uma consultoria de tecnologia baseada em Londres. Ela faz parcerias com empresas para apoiar necessidades de tecnologia, da estratégia à entrega. Oferece desenvolvimento de software sob medida, consultoria de transformação digital, aumento de time e suporte técnico contínuo, ajudando clientes a construir produtos e escalar capacidades rapidamente com profissionais experientes.

A função
Como pessoa engenheira full stack com experiência em .NET e Vue, você atuará de forma prática na construção de aplicações web, trabalhando tanto no front quanto no back-end. As responsabilidades incluem desenhar, desenvolver e refinar aplicações web usando .NET 6+, EF Core, JavaScript, PrimeFaces e Vue.js 2/3 para criar interfaces envolventes e fáceis de usar.

Responsabilidades:
- Desenvolver e manter soluções full stack
- Projetar aplicações escaláveis, seguras e fáceis de manter
- Implementar TDD, testes automatizados e code reviews
- Criar e manter documentação técnica
- Colaborar com engenheiros e stakeholders em Agile/Scrum
- Investigar e resolver problemas de clientes

Requisitos:
- 2+ anos de experiência em desenvolvimento de software, idealmente web
- Não exige formação específica se você demonstrar as habilidades técnicas necessárias
- Inglês profissional avançado (falado e escrito)
- Proficiência em: ASP.NET Core, JavaScript, SQL Server, Vue (ou outro framework front-end), Git, TDD

Diferenciais:
- TypeScript
- Azure
- DevOps
- CSS/SCSS

Condições:
- Bônus anual
- Plano de saúde privado
- Previdência/pensão
- Trabalho flexível
- Bons equipamentos

Posição majoritariamente remota no Reino Unido, com 1 dia por mês no escritório do cliente em Bloxham.
