# Frontend Developer (Vue.js) - micro1

A micro1 divulga uma vaga para Frontend Developer com foco em Vue.js para atuar de forma remota como contractor em um time de cliente. A função envolve desenvolver e manter soluções frontend escaláveis, trabalhar com Vue.js e React, transformar wireframes em interfaces responsivas, criar componentes reutilizáveis, otimizar performance e colaborar com engenharia backend em um ambiente distribuído.

Os requisitos incluem experiência avançada com JavaScript moderno, vivência profissional com Vue.js em aplicações complexas, experiência prática com React, domínio de CSS ou SCSS, ferramentas de build e versionamento e comunicação escrita e verbal muito forte. Experiência com times ágeis, TypeScript e aplicações acessíveis aparece como diferencial.
