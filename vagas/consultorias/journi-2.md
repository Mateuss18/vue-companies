# Fullstack Developer (Vue/.Net) - Fully Remote (Europe) - Cliente

Sobre
A Journi é uma consultoria de tecnologia baseada em Londres. Ela faz parcerias com empresas para apoiar necessidades de tecnologia, da estratégia à entrega. Oferece desenvolvimento de software sob medida, consultoria de transformação digital, aumento de time e suporte técnico contínuo, ajudando clientes a construir produtos e escalar capacidades rapidamente com profissionais experientes.

A função
Como pessoa engenheira full stack com experiência em .NET e Vue, você atuará de forma prática na construção de aplicações web, trabalhando tanto no front quanto no back-end. Você trabalhará de perto com o time do cliente para entregar uma solução que atenda aos requisitos.

Responsabilidades
- Desenvolver e manter aplicações web usando .NET e Vue.js
- Colaborar com o time de desenvolvimento para desenhar soluções eficientes e escaláveis
- Participar de code reviews e testes unitários
- Pesquisar e propor soluções tecnológicas inovadoras

Requisitos
- Experiência comprovada em desenvolvimento web com .NET 6+ e Vue 2/3
- Conhecimento sólido de tecnologias relacionadas como Web API, CQS, MediatR e arquiteturas orientadas a eventos
- Conhecimento sólido de HTML, CSS e TypeScript
- Capacidade de trabalhar de forma autônoma e em equipe

O que oferecemos
- Ambiente totalmente flexível e remoto
- Contrato de 12 meses (como contractor)
- Remuneração competitiva
