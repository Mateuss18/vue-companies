# Full Stack Developer - Java & Vue.js - Rextart

A Rextart está contratando Full Stack Developers middle e senior com Java e Vue.js para um projeto estruturado com foco em microserviços. A atuação combina desenvolvimento e manutenção de aplicações enterprise, com responsabilidades no backend em Java, design e evolução de microserviços, uso de Quarkus, integração com banco via PL/SQL e desenvolvimento de componentes frontend com Vue.js.

A descrição pede experiência com Java 8 até 17 ou 21, microserviços, Quarkus, PL/SQL, Maven, Git e trabalho colaborativo em projetos complexos. A vaga é full remote na Itália e a empresa destaca crescimento profissional, formação contínua, pacote de benefícios e atividades de integração.
