# Frontend Developer | Vue & Nuxt (Inglés) - Hasten Group

Na Hasten Group estamos selecionando um(a) Frontend Developer com pelo menos 3 anos de experiência em desenvolvimento de componentes web com Vue.js e Nuxt.js para atuar com um grande cliente, em um projeto internacional para o setor hoteleiro.

Requisitos mínimos:
- Experiência de pelo menos três anos como Frontend Developer
- Experiência sólida em desenvolvimento de componentes web
- Domínio de Vue.js 3 + Nuxt.js, sendo imprescindível ter pelo menos 2 a 3 anos de experiência em ambos
- Experiência com Angular e/ou React
- Conhecimento de bibliotecas Bootstrap e Tailwind
- Experiência com testes, performance e acessibilidade
- Desejável experiência com Git para controle de versões, XCode e Postman
- Desejável experiência com metodologias ágeis, como Scrum
- Inglês B2/C1 para trabalhar no idioma
- Residência na Espanha, com disponibilidade para ir pontualmente ao cliente em Madri
