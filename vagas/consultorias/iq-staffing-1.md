# Frontend Developer - IQ Staffing

Sobre a vaga

Estamos buscando um(a) Front-End Developer para atuar em um time Scrum dentro do domínio de riscos não financeiros de uma grande organização de serviços financeiros.

Você trabalhará de perto com desenvolvedores frontend e backend para projetar, desenvolver e manter aplicações customizadas de gestão de casos usadas por áreas internas como Compliance, SIM e Operações.

Todas as aplicações são desenvolvidas e mantidas integralmente pelo time e executadas em ambiente Microsoft Azure.

O time de desenvolvimento é relativamente grande, mas opera em subgrupos rotativos menores, o que favorece colaboração próxima, compartilhamento de conhecimento e senso de dono entre projetos.

Responsabilidades

- Desenvolver e manter aplicações frontend modernas com Vue 3
- Colaborar com desenvolvedores backend, product owners e stakeholders em um ambiente ágil Scrum
- Contribuir em refinamentos e discussões de user stories
- Construir e evoluir aplicações internas de gestão de casos
- Implementar testes unitários e end-to-end
- Participar de code reviews e discussões técnicas
- Aplicar padrões de design e boas práticas de desenvolvimento
- Trabalhar com pipelines de CI/CD e infraestrutura em Azure

O que você traz

Soft skills
- Postura proativa, com forte senso de dono e responsabilidade
- Boa comunicação e colaboração
- Paixão por aprendizado contínuo
- Capacidade de contribuir ativamente em refinamentos
- Atuação como parceiro técnico de confiança dentro do time
- Interesse por tendências do setor e tecnologias modernas
- Capacidade de inspirar colegas e trazer novas ideias

Competências técnicas
- Vue 3, com experiência adicional em outros frameworks JavaScript como diferencial
- HTML, CSS e JavaScript sólidos
- Experiência com frameworks de testes unitários e end-to-end, como Vitest e Playwright
- Experiência com Git ou Azure Repos
- Entendimento de padrões de design
- Experiência em times Agile/Scrum
- Familiaridade com infraestrutura de CI/CD em Azure DevOps

Stack utilizada
- Vue 3
- Azure Cloud
- Azure DevOps
- Git / Azure Repos
- Vitest
- Playwright
- Pipelines de CI/CD

Se você gosta de construir aplicações modernas em Vue e trabalhar em um ambiente em nuvem, queremos te conhecer.
