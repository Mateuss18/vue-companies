# Senior Vuejs Developer - VTI

Na VTI, a vaga busca alguém para desenvolver e otimizar aplicações front-end com React ou Vue usando TypeScript, criar componentes complexos como árvores, canvas e painéis de inspeção, transformar grandes volumes de dados em estruturas utilizáveis e colaborar de perto com backend, QA e produto para entregar funcionalidades com alta qualidade.

O perfil pedido exige mais de 6 anos de experiência com React ou Vue e TypeScript, domínio de reatividade, hooks, gerenciamento de estado com Redux, Zustand ou Pinia, experiência com interfaces complexas, transformação de dados aninhados, otimização de performance, arquitetura de CSS e mentalidade forte de system design. A vaga também destaca que o trabalho é do tipo bodyshop.

A empresa oferece bônus por desempenho e projeto, plano de saúde, viagens e teambuildings, aulas de japonês, treinamentos para certificações em cloud e AWS e jornada de segunda a sexta.
