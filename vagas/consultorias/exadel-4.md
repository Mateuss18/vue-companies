# Associate Full Stack Software Engineer (Java, Vue.js) - Exadel

A Exadel busca uma pessoa Associate Full-Stack Software Engineer para trabalhar com Java e Vue.js em soluções modernas e escaláveis. O projeto atende um cliente líder em soluções para o ciclo de vida de veículos, conectando seguradoras, oficinas, fornecedores de peças e parceiros de dados para melhorar produtividade e experiência do usuário final.

As responsabilidades incluem colaborar com times e usuários finais, desenvolver software e testes automatizados, cumprir entregas no prazo, participar de discussões técnicas e identificar oportunidades de refatoração e otimização. A vaga pede experiência com Java/J2EE, GitHub, Maven, AWS, SoapUI, Spring Boot, HTML, JavaScript, Vue.js, Spring, Hibernate, PostgreSQL ou MySQL, além de inglês upper-intermediate.
